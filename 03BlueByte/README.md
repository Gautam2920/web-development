# BlueByte

**Live Demo:**  <https://blue-byte-five.vercel.app/>

A winter-toned retro calculator that clicks, pops, and behaves like it
actually belongs on your desktop.

------------------------------------------------------------------------

## Features

- Clean arithmetic engine with repeat-last support
- Keyboard input support
- Light / Dark theme toggle
- Subtle hover feedback
- Result "pop" animation
- Dual-tone sound feedback
- Copy-to-clipboard
- Persistent theme, sound, and last calculation
